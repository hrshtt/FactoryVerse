# System Prompts

This directory contains all versions of the FactoryVerse agent system prompts.

## Current Version: v3

**Recommended**: `factoryverse-system-prompt-v3-core.md`

### Base Template
- `factoryverse-system-prompt-v3.md` - Base template with placeholders (254 lines)
  - Contains strategic guidance and Factorio wisdom
  - Uses XML structure for organization
  - Requires documentation injection via placeholders

### Assembled Versions
- `factoryverse-system-prompt-v3-core.md` - **Core version without examples** (690 lines)
  - Base prompt + DSL docs + DuckDB docs
  - Recommended for production use
  - Examples available separately in `examples/` directory
  
- `factoryverse-system-prompt-v3-with-examples.md` - Full version with examples (1,350 lines)
  - Includes all 5 code examples inline
  - Useful for testing and comprehensive reference
  - Larger token count

## Previous Versions

- `factoryverse-system-prompt-v2.md` - Previous version (1,350 lines)
  - Monolithic structure with embedded API docs
  - Good strategic content, harder to maintain
  
- `factoryverse-system-prompt.md` - Original version (v1)
  - Legacy reference

## Assembly Script

To regenerate assembled versions:

```bash
# Core version (without examples)
python3 /tmp/assemble_prompt_v2.py

# Full version (with examples)
python3 /tmp/assemble_prompt_v2.py --with-examples
```

## Documentation Dependencies

The v3 prompts require these auto-generated files:
- `dsl_documentation.txt` - Generated by `scripts/documentation_DSL.py`
- `duckdb_documentation.txt` - Generated by `scripts/documentation_duckdb.py`

## Example Code

Runnable example files are in `examples/`:
- `01_basic_query_and_walk.py`
- `02_mining_resources.py`
- `03_placing_entities.py`
- `04_automation_setup.py`
- `05_full_workflow.py`

These demonstrate key patterns and can be included in the prompt or referenced separately.
